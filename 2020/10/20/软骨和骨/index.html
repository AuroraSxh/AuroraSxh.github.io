软骨和骨

软骨

构成

软骨组织

软骨细胞
  • 骨祖细胞——软骨组织干细胞
  • 成软骨细胞
  • 软骨细胞
软骨基质
  • 固态
  • 无血管、神经、淋巴结
  • 营养依靠渗透作用，效率较低（修复慢）

根据纤维类型区分软骨组织

共同点

  • 细胞类型相通
  • 同源细胞群：从软骨周边向深部，软骨细胞渐渐长大分裂，常成群分布，每个细胞都有各自的软骨囊，这些细胞是由一个软骨细胞分裂而来，称为同源细胞群。
  • 软骨组织内无血管

透明软骨（可钙化）

  • 分布：肋、关节面、鼻、喉、气管、支气管、骺板、早期胚胎骨架

  • 细胞

    • 软骨细胞
    • 成软骨细胞

  • 基质

    • 胶原原纤维
    • 硫酸软骨素（HE染色蓝染）
    • 硫酸角质素
    • 透明质酸

  • 有软骨膜

  • 功能

    • 对抗压力、减少摩擦，提供结构支持
    • 软骨内成骨，长骨生长

纤维软骨

  • 分布：椎间盘、关节盘、半月板、耻骨联合、肌腱和韧带插入部

  • 细胞

    • 软骨细胞
    • 成纤维细胞

  • 基质

    • 交叉垂直或平行的胶原纤维束

  • 无软骨膜

  • 功能

    • 对抗压力和应力，吸收震荡

弹性软骨

  • 分布：耳廓、咽鼓管、会厌、喉

  • 细胞

    • 软骨细胞
    • 成软骨细胞

  • 基质

    • 弹性纤维

  • 有软骨膜

  • 功能

    • 提供柔韧的结构支持

软骨的生长

  • 间充质细胞密集成团，称软骨形成中心

  • 间充质细胞分化为成软骨细胞，分泌基质和纤维

  • 软骨形成中心周围间充质分化为软骨膜。

  • 继续生长

    • 间质生长（软骨内生长，软骨从内部增大
    • 外加生长（软骨膜下生长，软骨从表面向外扩大）

骨

包含血管和神经（与软骨区分！)

骨组织（多种细胞和大量钙化的细胞间质（又称骨基质）组成）

骨基质

  • 本质为钙化的细胞外基质

  • 组成

    • 有机成分（大量胶原纤维）：无机成分（骨盐）——35%：65%
    • 胶原纤维：无定形基质——95%：5%

  • 特征

    • 胶原纤维高度有规律地排列成层，因此又称骨板
    • 骨盐沉积在每层胶原纤维上，再由基质粘合而成薄板状结构
    • 每一层骨板厚约3-7μm
    • 同一层骨板内纤维相互平行，相邻骨板内纤维相互垂直

骨细胞

骨原细胞——骨组织的干细胞

  • 分布：骨外膜和骨内膜贴近骨质处

  • 光镜：

    • 较小，呈梭形
    • 胞核椭圆形
    • 胞质较少，弱嗜碱性（HE浅蓝)

  • 功能：

    • 骨组织生长或重建时，分裂、分化成骨细胞
    • 静止时称为骨被覆细胞
成骨细胞

  • 分布：骨组织表面，紧密排列成一层

  • 光镜

    • 细胞较大，呈矮柱状或椭圆形
    • 胞核圆形，一个，位于细胞游离端，核仁明显
    • 胞质强嗜碱性（HE深蓝）

  • 电镜结构：

    • 表面有细小突起骨质侧突起伸入骨基质表面的骨小管内，与骨细胞突起相连接
    • 胞质：含大量rER和发达的高尔基复合体

  • 功能

    • 分泌骨基质有机成分（胶原纤维和基质）——（未有钙盐沉积时为类骨质）
    • 释放细胞质内基质小泡，小泡内钙沉积于类骨质上形成骨基质
    • 成骨细胞及其突起被埋入骨基质内即成为骨细胞，形成骨陷窝和骨小管
骨细胞

  • 分布：骨板内和骨板间

  • 光镜结构

    • 扁平椭圆多突型
    • 胞核卵圆形，染色质多
    • 胞质弱嗜碱性
    • 所占据空间为骨陷窝，突起所占据的空间成为骨小管，各骨陷窝依靠骨小管彼此沟通

  • 电镜结构

    • 相邻骨细胞突起借缝管连接相互连接
    • 胞质内含高尔基复合体、散在rER和少量线粒体

  • 功能

    • 营养、更新和维持骨基质
    • 维持血钙恒定
破骨细胞

  • 分布：数量少，分布于骨吸收面的凹陷处

  • 光镜结构

    • 巨大，圆形或椭圆形
    • 骨基质侧有褶皱缘，周围有环形的两区
    • 多核，一般有6-50个胞核，由多细胞融合而成
    • 胞质嗜酸性，泡沫状

  • 电镜结构

    • 褶皱缘为许多不规则的微绒毛
    • rER丰富，高尔基复合体发达、线粒体和溶酶体多
    • 环形亮区含有大量的微丝，亮区与骨基质表面紧密相贴，封闭内侧构成溶骨微环境

  • 功能：具有很强的重吸收骨的能力

长骨的结构

骨松质

  • 分布于长骨的骨骺部和骨干内侧
  • 大量片状和针状骨小梁相互连接，组成多孔的网状结构
  • 骨小梁之间的孔内充满了红骨髓和血管
  • 骨小梁由成层平行排列的骨板和骨细胞构成

骨密质

  • 分布于长骨骨干和骨骺表层

环骨板

外环骨板
  • 较厚（10-40）层，环绕、平行排列于骨干表面，外表面与骨膜紧密相接，较整齐
内环骨板
  • 较薄（仅由数层骨板组成），居于骨干的骨髓腔面，内表面衬以骨内膜，后者与被覆于骨松质表面的骨内膜相接续。不如外环骨板平整
穿通管（福尔克曼管
  • 骨膜的血管、神经横向穿行于内、外环骨板形成的管道，与纵向排列的中央管相通。管中血管与骨髓血管相通连。

骨单位（哈弗斯系统）

  • 内、外环骨板之间

  • 长骨干的结构和营养单位，长筒型

  • 中央管（哈弗斯管

    • 位于骨单位的中央

    • 内有血管、神经及少量结缔组织

  • 哈弗斯骨板

    • 以中央管为中心呈同心圆排列，约10-20层

粘合线

  • 含骨盐多、纤维少的骨基质
  • 哈弗斯系统最外层具粘合线与相邻骨板相隔
  • 骨单位最外层的骨小管并不与相邻骨单位的骨小管相连

间骨板

  • 填充在骨单位之间的一些半环形、不规则骨板；是上一代为被吸收的骨单位或外环骨板的残留部分；其中除骨陷窝及骨小管之外，无其他管道

骨膜

  • 除关节面外、骨的内、外表面均被覆一层致密的结缔组织，分别为骨外膜和骨内膜

    骨外膜

  • 位于长骨外表面，分为两层，外层较厚，由致密结缔组织所组成；内层较薄，结缔组织较疏松，纤维少，细胞多，含有较多的骨原细胞或成骨细胞

骨内膜

  • 位于骨髓腔面、骨小梁表面、中央管及穿通管内表面；纤维少，骨原细胞常排列成一层，可分化为成骨细胞

骨生长及再生

骨发生

  • 由胚胎时期间充质分化而来

    过程

  • 成骨细胞形成——间充质细胞分裂增殖、分化为骨原细胞、成骨细胞

  • 类骨质形成——成骨细胞产生纤维和无定形基质，形成类骨质

  • 骨基质形成——类骨质具有沉积骨盐的特性，骨盐沉着后，类骨质骨化为骨基质。即一层新骨板出现，骨陷窝和骨小管形成

  • 骨细胞形成——成骨细胞之间的距离逐渐加大，突起加长，包埋入骨基质后成为骨细胞，骨组织发生完成

膜内成骨

  • 额骨、顶骨、面骨及锁骨等扁骨
  • 间充质分化为原始结缔组织膜–>间充质细胞分化为骨祖细胞–>骨祖细胞分化为骨细胞，成骨细胞形成骨组织

软骨内成骨

  • 四肢骨与躯干骨

  • 过程：

    • ①形成软骨雏形 ：间充质细胞聚集分化为成骨祖细胞，成骨祖细胞分化为软骨细胞产生软骨基质，形成软骨雏形。
    • ②形成骨领 ：在软骨雏形中段，软骨膜内的骨祖细胞分化 为成骨细胞，在软骨膜下形成领圈状薄层原始骨组织，即骨领，包绕软骨雏形中段。
    • ③形成初级骨化中心与骨髓腔 ：软骨雏形中央软骨细胞凋亡，软骨基质钙化。破骨细胞分解软骨，成骨细胞成骨，以钙化软骨基质为中轴，表面附以骨组织的 过渡型骨小梁 。开始出现过渡型骨小梁的地方即为初级骨化中心，之间为 初级骨髓腔 。 （有血管 ，有破骨细胞间充质细胞在此分化成网状细胞，形成网状组织，繁衍为骨髓。
    • ④形成次级骨化中心与骨骺 ：骨干两端的软骨中央出现次级骨化中心。骨骺与骨干之间的软骨层称骺板。

骨生长

  • 影响因素

    • 遗传、激素、维生素、其他活性物质、营养状况、体育锻炼、应力作用

  • 软骨增长的速度和软骨破坏的速度保持平衡状态，所以骺板的厚度相对恒定

  • 17-20岁骺板完全被骨质所替换，骨骺线形成，骨停止伸长。

  • 骨增粗由骨膜向故表面不断形成外环骨板，25-30岁停止

骨再生

  • 骨组织再生能力强
