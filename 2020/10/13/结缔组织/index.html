疏松结缔组织

- 特点

    - 细胞种类多
    - 纤维数量少，排列稀疏（疏松）

- 分布

    - 广泛，器官、组织、细胞之间

- 功能

    - 连接
    - 支持
    - 营养
    - 防御
    - 保护
    - 修复

- 组成

    - 细胞

        - 成纤维细胞

            - <font color = red>主要细胞！</font>
            - 光镜结构

                - 扁平，多突起
                - 细胞核较大，扁卵圆形，<font color = red>染色质颗粒较小且着色浅，核仁明显</font>
                - 胞质丰富，<font color = red>弱碱性</font>

            - 电镜结构

                - 微绒毛+粗短突起
                - 富含rER和游离多核糖体，高尔基体发达

            - 功能

                - 合成和分泌形成纤维与基质的蛋白

        - 纤维细胞

            - 静止状态的成纤维细胞

        - 巨噬细胞

            - 吞噬能力强，数量多，分布广
            - 光镜结构

                - 形态随功能状态改变而改变，<font color = red>常见伪足</font>
                - 核较小，<font color = red>卵圆形或肾形，偏心位，着色深，核仁不明显</font>
                - 胞质丰富，<font color = red>多呈嗜酸性，常含空泡和吞噬颗粒</font>

            - 电镜结构

                - 表面有褶皱、微绒毛和球形隆起
                - 胞质内含

                    - 初级溶酶体
                    - 次级溶酶体
                    - 吞噬体
                    - 吞饮小泡
                    - 残余体

                - 较多rER，较多微丝微管

            - 功能

                - 趋化运动

                    - 趋化性
                    - 趋化因子

                - 吞噬
                - 分泌
                - 参与免疫调节与应答

        - 浆细胞

            - 光镜结构

                - 卵圆形、圆形
                - 核圆形，偏居一侧，染色质 **<font color = red>辐射状（车轮状）</font>**
                - 胞质丰富，<font color = red>嗜碱性</font>，核旁有浅染区

            - 电镜结构

                - 表面平滑
                - rER、游离多核糖体丰富、高尔基体发达
                - 中心体位于浅染区

            - 功能

                - 合成和分泌免疫球蛋白（抗体+细胞因子）
                - 参与体液免疫、调节炎症反应

        - 肥大细胞

            - 光镜结构

                - 较大、圆形/卵圆形
                - 核小、<font color = red>圆形/卵圆形，多位于中央，染色质分布在核的边缘</font>
                - <font color = red>嗜碱性颗粒</font>、易溶于水、**<font color = red>异染性</font>**

            - 电镜结构

                - 颗粒大小不一，圆形/卵圆形
                - 膜包裹，内部螺旋状

            - 功能

                - 参与<font color = red>过敏反应（组胺，白三烯）</font>
                - 抗凝（肝素）
                - 抗过敏（嗜酸性粒细胞趋化因子）

        - 脂肪细胞（合成、储存，脂类代谢）

            - 单泡
            - 多泡

        - 白细胞

            - 变形作用
            - 防御

        - 未分化的间充质细胞

            - 保持分化潜能

    - 胞外基质

        - 纤维

            - 胶原纤维（韧性）

                - 光镜结构

                    - **<font color = red>嗜酸性</font>**
                    - 粗细不等，波浪形，互相交织

                - 电镜结构

                    - 67nm周期性横纹

                - 特性

                    - 新鲜时白色
                    - <font color = red>胶原原纤维+黏合质</font>
                    - Ⅰ型和Ⅲ型胶原蛋白在胞外聚合（成纤维细胞分泌）

            - 弹性纤维（弹性）

                - 光镜结构

                    - <font color = red>HE染色与胶原纤维相似，醛复红或地衣红染色为紫色/棕褐色</font>
                    - 与胶原相比更细更直
                    - 光滑，断端卷曲，有分支，交叉成网

                - 电镜结构

                    - 核心部分电子密度低，由<font color = red>弹性蛋白</font>组成
                    - 外覆盖微原纤维（原纤维蛋白构成）

                - 理化特性

                    - 新鲜时黄色
                    - 强烈日照易断裂

            - 网状纤维

                - 光镜结构

                    - 细、分支多、网状
                    - <font color = red>嗜银性，PAS阳性（糖）</font>

                - 电镜结构

                    - 67nm周期性横纹，Ⅲ型胶原蛋白

                - 分布+功能

                    - 参与构成基膜
                    - 构成网状组织的微细支架
                    - 多分布于结缔组织与其他组织交界处

        - 基质

            - 蛋白聚糖

                - 糖胺聚糖

                    - 硫酸化
                    - 非硫酸化

                        - 透明质酸（长链大分子）

                - 蛋白质

                    - 核心蛋白
                    - 结合蛋白

                        - 连接于透明质酸分子主干上形成分子筛（屏障作用）

            - 多黏糖蛋白

                - 纤维黏连蛋白

                    - 成纤维细胞与上皮细胞
                    - 细胞识别、黏附、迁移、增殖

                - 层粘连蛋白

                    - 上皮细胞与内皮细胞
                    - 上皮细胞与基膜的黏附

                - 软骨黏连蛋白

        - 组织液

            - 产生：毛细血管动脉端
            - 回流：毛细血管静脉端，毛细淋巴管
            - 组织和细胞赖以生存的内环境
            - 平衡

                - 水肿
                - 脱水

致密结缔组织

- 规则致密结缔组织

    - 胶原纤维大量密集顺受力方向排列
    - 肌腱、韧带……

- 不规则致密结缔组织

    - 粗大胶原纤维彼此交织
    - 真皮、硬脑膜、器官被膜……

- 弹性组织

    - 弹性纤维为主，之间有细胶原纤维
    - 形态

        - 平行排列成束
        - 编织成膜

  • 脂肪组织

    • 黄色脂肪组织

      • HE染色空泡
      • 大脂滴
      • 单泡脂肪细胞

    • 棕色脂肪组织

      • 多泡脂肪细胞
      • 毛细血管丰富
      • 丰富小脂滴

  • 网状组织

    • 网状细胞

      • 星形，细胞突起链接成网，胞核大，圆形/卵圆形，着色浅，核仁明显，rER发达

    • 网状纤维

    • 功能

      • 造血组织与淋巴器官的支架
      • 血细胞发生和淋巴细胞发育微环境
