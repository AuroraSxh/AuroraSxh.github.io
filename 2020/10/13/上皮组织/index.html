上皮组织

特点

  • 细胞多，间质少，细胞排列紧密

  • 细胞成层排列，有单层和复层之分

  • 单层上皮细胞有极性，有游离面和基底面，结构有差异

  • 上皮组织内无血管，有较多神经末梢（营养依靠渗透作用）

    特殊结构

游离面（与外界接触）

  • 微绒毛（光镜不可见）

    • 上皮细胞游离面伸出的细小指状突起

    • 微绒毛发达说明生物活性高

    • 结构

      • 顶部存在致密物质
      • 平行排列的微丝连接于终末网

    • 功能

      • 扩大细胞表面积
      • 参与细胞吸收物质

  • 纤毛（光镜可见）

    • 细胞游离面伸出的能摆动的较长的突起

    • 9+2微管结构（9组二联体微管包裹2根中央鞘包围的中央微管）

    • 功能

      • 方向节律性摆动
      • 将上皮表面的分泌物等向一定方向推送

  • 细胞衣

    • 细胞膜表面的复合糖（PAS positive?）
    • 游离面细胞衣显著
    • 黏着、支持、保护、物质交换、识别（糖蛋白）

侧面

  • 黏着物质

  • 嵌合

  • 细胞连接

    • 紧密连接

      • 细胞侧面顶端
      • 相邻细胞细胞膜形成2-4个点状融合，融合处细胞间隙消失，蛋白质交错成网格形状的嵴，嵴彼此贴合形成封闭索
      • 呈条带环绕细胞
      • 阻挡物质穿过，起屏障作用

    • 中间连接

      • 紧密连接下方，环绕上皮细胞的顶部

      • 结构

        • 相邻细胞间有12-20nm的间隙
        • 微丝形成的终末网（胞质）–锚定蛋白与钙粘蛋白胞内部分相连（胞质面）–钙粘蛋白胞外部分与另一细胞钙粘蛋白胞外部分相连–……

      • 黏着，维持细胞形状，传递收缩力

    • 桥粒

      • 斑状，纽扣状

      • 结构

        • 细胞间隙20-30nm,间隙中央有致密的中间线，由丝状物质交织而成。细胞膜胞质面有桥粒斑（附着板），中间丝附着在桥粒斑上，固定支持

      • 分布于易受摩擦的皮肤、食管的上皮中

      • 非常牢固，固定、支持

    • 缝隙连接

      • 结构

        • 细胞间隙3nm,规律分别连接小体（6个连接蛋白分子，中央2nm直径管腔），聚集成斑状
        • 连接小体对接，管腔通连，细胞间直接交流

      • 作用

        • 传递化学信息
        • 电冲动

基底面

  • 基膜（上皮基底面与深部结缔组织间的一层薄膜，PAS阳性，为上皮组织和结缔组织共同分泌的产物）

    • 组成

      • 透明板——电子密度较低的薄层
      • 基板——上皮细胞分泌，丝状物质+无定形基质
      • 网板——成纤维细胞分泌，网状纤维+基质

    • 功能

      • 对上皮细胞的支持、连接、固着作用
      • 细胞增殖、分化迁移、细胞代谢、信号转导
      • 选择性通透作用（半透膜），调控上皮组织与深部结缔组织的物质交换

  • 质膜内褶（上皮细胞基底面的细胞膜垂直折向胞质形成的许多内褶）

    • 含大量线粒体（物质转运功能发达）
    • 扩大细胞基底面的表面积，增强离子和水分子的快速转运
    • 分布于肾小管（水与离子的重吸收）

  • 半桥粒

    • 上皮细胞基底面
    • 质膜内有桥粒斑，角蛋白丝附着
    • 将上皮细胞固定在基膜上

分类

被覆上皮

  • 单层上皮

    • 单层扁平上皮

      • 外观

        • 表面观

          • 多边形，核扁圆形，位于中央，细胞边缘呈锯齿状，相邻细胞彼此嵌合

        • 垂直切面观

          • 细胞核扁平，含核部分略厚，其余胞质非常少

      • 分类

        • 内皮

          • 衬于心脏、血管、淋巴管腔面的单层扁平上皮称为内皮
          • 表面光滑，有利于液体流动与物质交换

        • 间皮

          • 分布在胸膜、腹膜和心包膜表面的单层扁平上皮称为间皮
          • 表面光滑，减少器官之间的摩擦

    • 单层立方上皮

      • 外观

        • 表面观

          • 细胞呈多边形

        • 垂直切面观

          • 细胞呈立方形，核圆，位于细胞中央

      • 分布

        • 肾小管
        • 甲状腺滤泡

      • 功能

        • 吸收
        • 分泌

    • 单层柱状上皮

      • 外观

        • 表面观

          • 细胞呈多边形

        • 垂直切面观

          • 细胞呈柱状，核长椭圆形，长轴与细胞长轴平行，靠近细胞基底部

      • 分布

        • 胃肠、胆囊、子宫及输卵管腔面

        • 小肠

          • 小肠腔面的单层柱状上皮细胞游离面有密集排列的微绒毛，即纹状缘

        • 子宫、输卵管腔面

          • 具有纤毛，为单层纤毛柱状上皮

      • 功能

        • 吸收
        • 分泌

    • 假复层纤毛柱状上皮

      • 垂直切面观

        • 游离面至基底面可见柱状细胞、梭形细胞和锥体细胞
        • 似复层上皮，胞核不在一水平面但所有细胞的基底部均附着于基膜上

      • 分布

        • 呼吸道表面

        • 输精管和附睾管

          • 无杯状细胞，游离面无纤毛，称为假复层柱状上皮

      • 功能

        • 重要的保护功能

  • 复层上皮

    • 复层扁平上皮

      • 外观

        • 表层细胞呈鳞片状

        • 垂直切面观

          • 细胞形状不一
          • 靠近基膜的基底细胞呈立方形或矮柱状，分裂增殖能力强，新生细胞向表层移动
          • 中间层细胞呈多边形或梭形
          • 表面的数层细胞呈扁平状

      • 分布

        • 角化（浅层细胞胞核消失，胞质充满角蛋白）

          • 皮肤表皮

        • 未角化（浅层细胞仍有细胞核，角蛋白含量少）

          • 口腔、、食管、阴道

      • 功能

        • 较厚、耐摩擦、较强的机械保护作用
        • 阻止异物、病原体的入侵
        • 很强的再生修复能力

    • 复层柱状上皮

      • 深层为一层或多层多边形细胞
      • 浅层为排列整齐的柱状细胞
      • 眼结膜、男性尿道

    • 变移上皮

      • 形状层数的变化

        • 充盈——变薄，层数减少，细胞扁梭形
        • 收缩——变厚，层数增加，细胞立方形

      • 盖细胞胞质丰富，覆盖深面的几个细胞，可见两个核

        腺上皮

  • 按照细胞数量

    • 单细胞腺
    • 多细胞腺

  • 按照有无导管

    • 内分泌腺

    • 外分泌腺

      • 按照分泌的物质

        • 浆液腺

          • 浆液性腺泡——浆液性腺细胞组成，分泌蛋白
          • HE染色深，胞核圆

        • 黏液腺

          • 黏液性腺泡——黏液性腺细胞组成，分泌糖蛋白
          • HE染色浅，胞核扁

        • 混合腺

      • 按照分泌方式

        • 全浆分泌腺（细胞崩解-皮脂）
        • 顶浆分泌腺（胞吐）
        • 局浆分泌腺（胞吐或直接透过胞膜）

      • 按照形状

        • 单腺

          • 单管状腺
          • 单泡状腺
          • 单管泡状腺

        • 复腺

          • 复管状腺
          • 复泡状腺
          • 复管泡状腺

肌上皮

感觉上皮

杯状细胞

存在于单层柱状上皮与假复层纤毛上皮组织中

外观

  • 顶部膨大，底部狭细
  • 位于基膜上
  • 顶部胞质内充满黏原颗粒，颗粒中有PAS阳性蛋白
  • 核小，三角形或扁圆形，着色深，位于基底部

功能

  • 润滑和保护上皮
